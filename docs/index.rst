Welcome to active_mri's documentation!
==================================================
``active-mri-acquisition`` is a package that facilitates the application of reinforcement learning to
fastMRI data. In particular, ``active-mri-acquisition`` provides a gym-like environment for
simulating the execution of active cartesian k-space sampling policies, with easily configurable
user-provided reconstruction models.
